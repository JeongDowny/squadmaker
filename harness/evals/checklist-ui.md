## Tactical board
- 모바일에서 선수 카드가 겹치지 않는가

- 선수별 출전 요약이 실제 라인업과 모순되지 않는가

## Execution rule

- `docs/DESIGN.md`와 비교 검토
- 문제 발견 시 최소 1회 이상 수정 반복
