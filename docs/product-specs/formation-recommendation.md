# formation-recommendation.md

## Goal
감독이 각 쿼터 포메이션을 선택하면, 참가 선수 중에서 선발 11명을 자동 배치해 상위 3개 추천안을 제공한다.

## Inputs
추천 계산에 필요한 입력은 아래와 같다.

### Match-level inputs
- 경기 제목
- 경기 날짜
- 총 쿼터 수
- 각 쿼터 포메이션

### Player-level inputs
- 선수 이름
- 주포지션
- 부포지션 최대 2개
- 실력 점수 1~10 정수

### System rules
- 포지션 미지정 선수는 허용하지 않는다.
- 실력 점수 미입력 시 기본값 5를 사용한다.
- 후보는 최적화 대상이 아니다.
- 각 쿼터는 선발 11명으로 구성한다.

## Outputs
각 추천안은 아래 정보를 포함한다.

- 쿼터별 선발 11명 배치
- 선수별 총 출전 쿼터 수
- 쿼터별 평균 점수
- 주포지션 일치율
- 추천안 성격 설명
- 추천안별 가중치 정보

## Recommendation target
서비스는 포메이션 자체를 추천하지 않는다.

감독이 쿼터별 포메이션을 먼저 선택하고,
서비스는 그 포메이션 안에서 선발 11명 배치안을 추천한다.

## Scoring criteria
추천 계산은 아래 세 기준을 사용한다.

- 공평성
- 평균 점수 균형
- 포지션 적합성

세 기준은 모든 추천안에 공통으로 사용되지만, 추천안 A/B/C마다 실제 가중치는 다르다.

## Fairness
공평성은 선수별 총 출전 쿼터 수가 특정 선수에게 과도하게 몰리지 않도록 평가한다.

핵심 목표:
- 전체 참가 선수의 출전 기회를 최대한 균형 있게 분배한다.
- 특정 선수에게 출전이 지나치게 편중되지 않도록 한다.

## Quarter strength balance
평균 점수 균형은 각 쿼터 선발 11명의 평균 실력 점수 차이를 줄이는 방향으로 평가한다.

핵심 목표:
- 특정 쿼터에만 전력이 과도하게 몰리지 않도록 한다.
- 쿼터 간 평균 실력 점수 편차를 줄인다.

실력 점수 미입력 선수는 기본값 5를 사용한다.

## Position fit
포지션 적합성은 각 선수가 배치된 슬롯에 얼마나 적합한지를 점수화한다.

기본 점수:
- 주포지션: 1.0
- 부포지션: 0.7
- 인접 계열 포지션: 0.4
- 완전 비적합 포지션: 0.1

인접 계열 포지션의 구체적 정의는 `docs/DOMAIN.md`를 따른다.

## Recommendation variants

### Recommendation A
출전 쿼터 수 공평성을 가장 우선하는 안

가중치:
- 공평성: 60
- 평균 점수 균형: 25
- 포지션 적합성: 15

특징:
- 전체 선수의 출전 기회를 가장 균형 있게 배분하려고 시도한다.
- 평균 점수나 주포지션 일치율이 일부 손해를 볼 수 있다.

### Recommendation B
쿼터 평균 점수 균형을 가장 우선하는 안

가중치:
- 공평성: 35
- 평균 점수 균형: 45
- 포지션 적합성: 20

특징:
- 각 쿼터의 평균 전력 차이를 줄이는 데 집중한다.
- 공평성도 고려하지만, 일부 선수의 출전 기회가 덜 균등할 수 있다.

### Recommendation C
포지션 적합성을 가장 우선하는 안

가중치:
- 공평성: 30
- 평균 점수 균형: 20
- 포지션 적합성: 50

특징:
- 가능한 한 선수를 가장 익숙한 자리 위주로 배치한다.
- 공평성이나 평균 점수 균형보다 포지션 적합성을 더 중시한다.

## Supported formations
MVP 기본 지원 포메이션:
- 4-4-2
- 4-3-3
- 4-2-3-1
- 3-5-2
- 3-4-3
- 4-1-4-1
- 4-5-1
- 4-3-1-2
- 4-1-2-1-2
- 3-4-1-2

쿼터마다 서로 다른 포메이션을 선택할 수 있다.

## Recalculation after manual edits
사용자가 추천 결과를 수정하면 아래 값은 즉시 다시 계산한다.

- 선수별 총 출전 쿼터 수
- 현재 쿼터 평균 점수
- 전체 쿼터 평균 점수 분포
- 주포지션 일치율

## Failure cases
아래 상황에서는 추천이 제한될 수 있다.

- 특정 포메이션에 필요한 포지션 자원이 부족한 경우
- 참가 선수 수가 충분하지 않은 경우
- 포지션 편중이 심해 적절한 배치가 어려운 경우

이 경우 시스템은 아래를 보여줘야 한다.
- 현재 입력으로 배치가 어려운 이유
- 부족한 포지션 정보
- 가능한 대안 또는 수정 유도

## Non-goals
이 문서 범위에는 아래가 포함되지 않는다.

- 실시간 경기 중 교체 전략
- 전술 AI 코칭
- 상대 전술 대응
- 포메이션 자유 커스텀
- 후보 선수 최적화
