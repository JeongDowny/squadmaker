# formation-recommendation.md

## Goal
감독이 각 쿼터 포메이션을 선택하면, 참가 선수 중에서 선발 11명을 자동 배치해 출전 쿼터 수가 최대한 균등한 단일 추천안을 제공한다.

현재 v1은 출전 쿼터 수 공평성을 우선한다.
평균 점수 균형과 A/B/C 다중 추천안은 후속 확장 범위다.

## Inputs
추천 계산에 필요한 입력은 아래와 같다.

### Match-level inputs
- 경기 제목
- 경기 날짜
- 총 쿼터 수
- 각 쿼터 포메이션

### Player-level inputs
- 선수 이름
- 경기 참가 여부
- 주포지션
- 부포지션 최대 2개

### System rules
- 포지션 미지정 선수는 추천 후보로 사용하지 않는다.
- 경기 제외 상태의 선수는 추천 후보로 사용하지 않는다.
- 각 쿼터는 선발 11명으로 구성한다.
- 참가 가능 선수가 11명 미만이면 추천을 생성하지 않고 부족 이유를 표시한다.

## Outputs
추천안은 아래 정보를 포함한다.

- 쿼터별 선발 11명 배치
- 선수별 총 출전 쿼터 수
- 추천안 성격 설명
- 포지션 적합도 기반 슬롯 배치

## Recommendation target
서비스는 포메이션 자체를 추천하지 않는다.

감독이 쿼터별 포메이션을 먼저 선택하고,
서비스는 그 포메이션 안에서 선발 11명 배치안을 추천한다.
추천 이후에도 감독은 쿼터별 포메이션을 다시 선택할 수 있으며, 화면은 같은 참가 명단 기준으로 배치를 다시 계산한다.

## Scoring criteria
현재 v1 추천 계산은 아래 우선순위를 사용한다.

1. 선수별 출전 쿼터 수가 적은 선수를 우선한다.
2. 같은 출전 수라면 포지션 적합도가 높은 선수를 우선한다.
3. 같은 조건이라면 입력 순서를 유지한다.
4. GK가 비-GK 슬롯에 먼저 배치되지 않도록 후보를 제한한다.

## Fairness
공평성은 선수별 총 출전 쿼터 수가 특정 선수에게 과도하게 몰리지 않도록 평가한다.

핵심 목표:
- 전체 참가 선수의 출전 기회를 최대한 균형 있게 분배한다.
- 특정 선수에게 출전이 지나치게 편중되지 않도록 한다.
- 선수 목록에서 이 경기의 출전 쿼터 수를 바로 확인할 수 있게 한다.

## Position fit
포지션 적합성은 각 선수가 배치된 슬롯에 얼마나 적합한지를 점수화한다.

기본 점수:
- 주포지션: 1.0
- 부포지션: 0.7
- 인접 계열 포지션: 0.4
- 완전 비적합 포지션: 0.1

인접 계열 포지션의 구체적 정의는 `docs/DOMAIN.md`를 따른다.

## Deferred recommendation variants
A/B/C 다중 추천안은 현재 v1에서 제공하지 않는다.

후속 확장 시 고려할 수 있는 관점:
- 출전 쿼터 수 공평성 우선
- 쿼터 평균 점수 균형 우선
- 주포지션 일치율 우선

이 확장을 다시 도입할 때는 실력 점수 입력, 평균 점수 계산, 추천안 카드 UI, 저장 shape를 함께 재검토해야 한다.

## Supported formations
MVP 기본 지원 포메이션:
- 4-4-2
- 4-3-3
- 4-2-3-1
- 3-5-2
- 3-4-3
- 4-1-4-1
- 4-5-1
- 4-3-1-2
- 4-1-2-1-2
- 3-4-1-2

쿼터마다 서로 다른 포메이션을 선택할 수 있다.

## Recalculation after edits
사용자가 추천 후 포메이션이나 참가 여부를 수정하면 아래 값은 즉시 다시 계산한다.

- 쿼터별 선발 배치
- 선수별 총 출전 쿼터 수
- 포지션 슬롯 배치

## Failure cases
아래 상황에서는 추천이 제한될 수 있다.

- 참가 가능 선수 수가 11명보다 적은 경우
- 이름 또는 주포지션이 없는 선수가 많은 경우
- 포지션 편중이 심해 적절한 배치가 어려운 경우

이 경우 시스템은 아래를 보여줘야 한다.
- 현재 입력으로 배치가 어려운 이유
- 부족한 선수 정보
- 가능한 대안 또는 수정 유도

## Non-goals
이 문서 범위에는 아래가 포함되지 않는다.

- 실시간 경기 중 교체 전략
- 전술 AI 코칭
- 상대 전술 대응
- 포메이션 자유 커스텀
- 후보 선수 최적화
- 실력 점수 기반 평균 전력 균형 추천
