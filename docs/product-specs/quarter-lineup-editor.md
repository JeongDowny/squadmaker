# quarter-lineup-editor.md

## Goal
사용자가 추천된 쿼터별 선발 배치를 쉽게 확인하고, 필요 시 직접 수정하며, 수정 결과가 즉시 재계산되도록 한다.

## Scope
이 문서는 결과 확인 및 수동 편집 화면의 동작을 정의한다.

포함 범위:
- 쿼터 탭 전환
- 현재 쿼터 필드 배치 보기
- 선수 교체
- 선발/후보 변경
- 수정 후 재계산
- 모바일 편집 흐름

## Default view
기본 결과 화면은 한 번에 한 쿼터만 보여준다.

구성:
- 상단: 경기 정보
- 쿼터 탭
- 중앙: 현재 쿼터 필드 배치
- 하단 또는 측면: 출전 쿼터 카드 리스트
- 보조 정보: 평균 점수, 추천안 성격, 저장/내보내기 액션

## Quarter navigation
- 사용자는 탭으로 쿼터를 전환한다.
- 각 탭은 하나의 쿼터를 나타낸다.
- 탭 전환 시 해당 쿼터의 선발 배치와 관련 정보가 즉시 반영된다.

## Editable entities
사용자가 편집할 수 있는 대상은 아래와 같다.

- 특정 쿼터의 선발 선수
- 특정 슬롯의 배치 선수
- 선발과 후보 간 교체 결과

MVP 범위 밖:
- 포메이션 커스텀
- 경기 중 교체 시뮬레이션
- 후보 최적화

## Manual substitution flow
모바일 기준 기본 교체 방식은 탭 기반이다.

### Interaction steps
1. 현재 선발 또는 교체 대상 선수를 한 번 탭한다.
2. 시스템은 이 선수를 교체 대상으로 표시한다.
3. 두 번째로 다른 선수를 탭하면 교체를 실행한다.
4. 교체 후 관련 계산을 즉시 갱신한다.

### Selection state requirements
- 선택된 선수는 명확하게 강조 표시되어야 한다.
- 현재 어떤 상태인지 텍스트 또는 배지로 알려야 한다.
- 잘못 선택한 경우 취소가 쉬워야 한다.

## Desktop behavior
데스크톱에서는 드래그 앤 드롭을 지원할 수 있다.

원칙:
- 드래그 앤 드롭은 보조 수단이다.
- 드래그가 불편하거나 실패해도 탭/클릭 기반 교체가 가능해야 한다.
- 데스크톱과 모바일은 결과적으로 같은 수정이 가능해야 한다.

## Recalculation rules
수정 후 아래 항목은 즉시 다시 계산한다.

- 선수별 총 출전 쿼터 수
- 현재 쿼터 평균 점수
- 전체 쿼터 평균 점수 분포
- 주포지션 일치율

원칙:
- 재계산은 사용자가 따로 저장 버튼을 누르지 않아도 화면에 즉시 반영된다.
- 재계산 결과는 숫자만 바뀌는 것이 아니라, 상태 표시에도 반영된다.

## Bench handling
후보는 특정 쿼터에서 선발 제외된 경기 참가 선수다.

편집 시 원칙:
- 후보 리스트에서 선수를 선택해 선발과 교체할 수 있다.
- 선발에서 제외된 선수는 자동으로 해당 쿼터 후보가 된다.

## Validation rules
편집 과정에서 아래 상황을 감지해야 한다.

- 동일 쿼터 내 같은 선수가 중복 배치됨
- 포메이션 슬롯과 맞지 않는 배치가 과도하게 많음
- 저장 가능한 최종 구조가 아님

시스템은 가능한 경우 편집을 허용하되, 문제가 있으면 명확히 경고한다.

## Displayed metadata
편집 화면에서 최소 아래 정보를 함께 보여준다.

- 현재 쿼터 번호
- 현재 쿼터 포메이션
- 현재 쿼터 평균 점수
- 선수별 총 출전 쿼터 수
- 추천안 종류
- 저장 여부 또는 수정 여부

## Undo / cancel guidance
MVP에서 완전한 히스토리 undo는 필수는 아니지만 아래는 필요하다.

- 교체 대상 선택 취소
- 추천안 원본으로 되돌아가기
- 저장 전 현재 화면 기준으로 다시 계산하기

## Mobile principles
- 한 손 사용이 가능해야 한다.
- 너무 작은 터치 영역을 허용하지 않는다.
- 필드와 후보 리스트 사이 이동이 쉬워야 한다.
- 두 번 탭 교체 흐름이 헷갈리지 않게 상태를 명확히 보여줘야 한다.

## Success criteria
- 사용자가 쿼터별 선발 배치를 쉽게 확인할 수 있다.
- 교체가 직관적으로 가능하다.
- 수정 후 결과 변화가 즉시 이해된다.
- 모바일에서도 편집이 불편하지 않다.
