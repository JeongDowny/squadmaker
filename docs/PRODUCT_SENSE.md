# PRODUCT_SENSE.md

## One-line product definition
선수별 출전 시간과 쿼터별 실력 균형을 고려해 경기 전 선발 배치를 빠르게 짜게 도와주는 축구 경기 운영 서비스

## Who this product is for
이 서비스의 1차 타깃 사용자는 축구 동호회 회장 또는 감독이다.

이 사용자는 보통 아래 특징을 가진다.
- 포메이션과 역할 구분이 가능하다.
- 경기 전 선수 배치와 출전 분배를 직접 관리한다.
- 공평한 출전과 경기력 사이에서 균형을 잡아야 한다.
- 기존에는 메모장, 머릿속 계산, 카카오톡 줄글 또는 캡처 공유에 의존한다.

## Core problem
사용자는 경기 전 아래 문제를 동시에 해결해야 한다.

1. 선수별 출전 쿼터 수를 가능한 공평하게 분배해야 한다.
2. 각 쿼터의 평균 전력이 크게 차이나지 않아야 한다.
3. 포지션 적합성을 무시하면 배치 결과에 대한 신뢰가 떨어진다.
4. 경우의 수가 많아 수동으로 짜면 시간이 오래 걸린다.

즉, 단순히 11명을 뽑는 것이 아니라 여러 쿼터에 걸친 선발 배치를 빠르게 설계하는 것이 핵심 문제다.

## Existing workaround
현재 사용자는 주로 아래 방식으로 문제를 해결한다.

- 메모장에 선수 이름과 출전 순서를 적는다.
- 머릿속으로 출전 시간과 포지션을 계산한다.
- 카카오톡 줄글이나 캡처 이미지로 공유한다.

이 방식은 수정이 어렵고, 경우의 수를 빠르게 비교하기 힘들며, 공평성과 전력 균형을 동시에 맞추기 어렵다.

## Core value proposition
이 서비스는 감독이 경기 정보를 입력하면 바로 쿼터별 선발 배치를 추천해 준다.

핵심 가치는 아래와 같다.

- 선수 입력 후 빠르게 결과를 본다.
- 쿼터별 선발 배치를 자동으로 추천받는다.
- 선수별 총 출전 쿼터 수를 한눈에 확인한다.
- 쿼터별 평균 실력 점수를 비교할 수 있다.
- 수정 시 결과가 즉시 재계산된다.

## What the product does
이 서비스는 다음을 지원한다.

- 전체 등록 선수 풀 관리
- 경기 생성
- 경기 제목 및 날짜 입력
- 경기 참가 선수 선택
- 쿼터 수 입력
- 쿼터별 포메이션 선택
- 선발 11명 자동 배치 추천
- 상위 3개 추천안 제시
- 수동 교체
- 선수별 출전 쿼터 수 재계산
- 쿼터별 평균 점수 재계산
- 최종 수정본 저장
- PNG 이미지 내보내기

## What this product is not
이 서비스는 아래를 목표로 하지 않는다.

- 실시간 경기 분석
- 전술 AI 코칭
- 경기 중 교체 시뮬레이션
- 상세 전술 전략 추천
- 경기 데이터 자동 수집
- 협업 편집 또는 공유 링크 기반 공동 작업

## MVP definition
MVP에서는 아래를 반드시 제공한다.

1. 선수 카드 입력
2. 경기 제목 및 날짜 입력
3. 경기 참가 선수 선택
4. 쿼터 수 입력
5. 쿼터별 포메이션 선택
6. 쿼터별 선발 추천 결과
7. 추천안 A/B/C 제공
8. 선수별 출전 쿼터 수 표시
9. 쿼터별 평균 점수 표시
10. 수동 교체 후 실시간 재계산
11. 비로그인 임시 저장
12. 로그인 사용자 경기 저장
13. PNG 내보내기

## Success criteria for MVP
MVP는 아래 기준을 만족해야 한다.

- 사용자가 선수 입력 후 빠르게 첫 배치 결과를 볼 수 있다.
- 선수별 출전 쿼터 수가 지나치게 불공평하지 않다.
- 쿼터별 평균 점수 편차가 크게 벌어지지 않는다.
- 사용자가 추천 결과를 자유롭게 수정할 수 있다.
- 모바일에서도 핵심 기능을 동일하게 사용할 수 있다.

## Long-term direction
장기적으로 이 서비스는 축구 동호회 팀 운영 툴로 확장한다.

확장 가능성 예시:
- 경기 기록 축적
- 시즌별 선수 관리
- 경기별 히스토리 비교
- 포지션별 세부 능력치
- 9인 축구 등 경기 형식 확장
